# Tech Digests

Auto-generated AI/ML/RecSys digests, run via Claude Code RemoteTrigger.

- Trigger: https://claude.ai/code/scheduled (find "Tech Digest", hit Run)
- Output: `digests/<date>.md` + GitHub issue per run
- Figures: `assets/<date>/`

See `digest_workflow.md` for the workflow the remote agent follows.
